curl https://isms-backend-project.onrender.com/api/cat-photos

Read One

curl https://isms-backend-project.onrender.com/api/cat-photos/<id>

Update

curl -X PUT https://isms-backend-project.onrender.com/api/cat-photos/<id> -H "Content-Type: application/json" -d '{"indoorOutdoor":"outdoor","personality":["Playful"],"catUrl":"Updated cat entry"}'

Delete

curl -X DELETE https://isms-backend-project.onrender.com/api/cat-photos/<id>
